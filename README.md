# 8.5-40V-to-4.2V-Power-Board

A compact 2 × 3 cm power board with 8.5V–40V input and stable 4.2V output, featuring PowerPath management, automatic source switching, LiPo charging, and high efficiency. It is designed for automotive and embedded applications, especially ESP32, GPS, and GSM-based systems powered from vehicle batteries or a single LiPo cell.
